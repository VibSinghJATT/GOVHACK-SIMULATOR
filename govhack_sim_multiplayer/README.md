
# GOVHACK Simulator

A **cross-platform multiplayer terminal game simulator** inspired by hacking aesthetics.
This is a **game**, not a real hacking tool.

Author: VibSingh  
GitHub:
- https://github.com/VibSinghJATT
- https://github.com/Omgindia1234

## Features
- Multiplayer (LAN / Internet)
- Public server room: VSingh
- PvP mini-games
- AI Bosses & NPC factions
- XP, inventory, achievements
- Community mod support (sandboxed)
- Works on Linux, macOS, Windows, WSL

See `README_INSTRUCTIONS.md` to get started.

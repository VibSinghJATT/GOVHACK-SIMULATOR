
# Installation & Usage

## Linux / macOS / WSL
```bash
chmod +x install.sh
./install.sh
./run.sh
```

## Windows 10/11
```powershell
install.ps1
run.ps1
```

## Older Windows
Double click `install.bat`, then `run.bat`

## Server
```bash
cd central_server
python vsingh_server.py
```
